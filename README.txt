File List:
1. REtoNFA.py - Python script that converts postfix regular expressions into NFAs.

Compiler/Interpreter version:
- Python 3.11.9

Compile instructions:
- No compilation required.

Run instructions:
1. Open a terminal in the directory containing the REtoNFA.py script and the test file.
2. Use the following command to run the script with a test file:
    python REtoNFA.py example_test.txt
3. The program will output the NFAs generated from the regular expressions in the test file.
